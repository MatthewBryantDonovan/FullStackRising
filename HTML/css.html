We can apply style to the page using CSS this page is using 3 different ways to apply style

  • Inline CSS - look in the li tag where this text is to see the stylings
  • Internal CSS - Look above the body tag to see the style tag
  • External CSS - another file is being used to apply stylings see ./CSS/myStyles.css
